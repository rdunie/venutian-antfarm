# Guidance Registry

Fleet-wide index of active governance guidance (Tier 3 -- NICE TO HAVE). Each entry provides a summary and "what's in it for me?" relevance statement. Detail docs are linked when the guidance is too large to inline.

**Format per entry:**

```
### [Topic Title] ([Cx Role])
[One-line summary of the guidance]
**Relevance:** [Who needs to care and why]
**Detail:** [link to detail doc, if not inlined]
```

**Maintained by:** CKO (adds/removes entries when Cx roles publish or retire guidance).
